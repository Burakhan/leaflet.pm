Leaflet Polygon Management

A Leaflet Plugin For Creating And Editing Geometry Layers in Leaflet 1.0.
Draw, Edit, Drag, Cut, Adapt, and soon: Snap Features.

Getting Started

Install via npm

npm install leaflet.pm --save

Install via Bower

bower install leaflet.pm --save

Install Manually

Download the latest release here. Include leaflet.pm.min.js and leaflet.pm.css in your project.

Creating A Polygon

Click around on the map to create a polygon. Click on the first marker again to finish drawing.
The 'pm:create' event will be fired after finishing the polygon.

How it works

Add a button to the map that toggles drawing mode with

map.pm.addControls();

Enable drawing mode manually with

map.pm.enableDraw('Poly');

Disable drawing mode manually with

map.pm.disableDraw();

Editing A Polygon

Click on the polygon to toggle edit mode.
You can add new markers by clicking/dragging the transparent markers between the current ones.
Remove a marker with right click.

How it works

With 

polygon.pm.toggleEdit();
you can toggle the edit mode for any polygon on the map.
Put it whereever you want. For example inside a click handler on a button or the polygon itself.

Please see the readme for more code examples.

Editing A Layer Group

Same as above. The LayerGroup consists of Polygons and will just foward all events.
So when you toggle edit mode for a layerGroup, you're actually toggling edit mode for every
layer inside the group.

How it works

With 

layerGroup.pm.toggleEdit();
you can toggle the edit mode for any layerGroup on the map.

Please see the readme for more code examples.
